<h1 align="center">React Native Template: Modern</h1>

### Usage

When creating a new React Native project, use the command:

`react-native init --template modern`

### Includes

- [x] TypeScript
- [x] Prettier (with commit hook)
- [x] TSLint
- [ ] Jest
- [ ] Web support

### Requirements

- `npm i -g react-native-cli`
